# <span style="color: #98FF98;">Математические формулы</span>
## <span style="color: #ADD8E6;">В данном файле представлены формулы, которые используются для геометрических фигур:</span>

### <span style="color: #98FF98;">Функции для фигуры круг:</span>
#### Периметр:
```python
from geometric_lib import *

# perimeter(r)


Параметры:
    r (float) - радиус круга

Возвращаемое значение:
    print(perimeter(10))  # периметр(длина окружности)
```
---
#### Площадь:
```python
from geometric_lib import *

# area(r)

Параметры:
    r (float) - радиус круга

Возвращаемое значение:
    print(area(10))  # площадь окружности
```
---
### <span style="color: #ADD8E6;">Функции для фигуры квадрат:</span>
#### Периметр:
```python
from geometric_lib import *

#perimeter(a)

Параметры:
    a (int): длина стороны квадрата

Возвращаемое значение:
    print(perimeter(10)) #периметр квадрата
```
---
#### Площадь:
```python
from geometric_lib import *

#area(a)

Параметры:
    a (int): длина стороны квадрата

Возвращаемое значение:
    print(area(10)) #площадь квадрата
```
